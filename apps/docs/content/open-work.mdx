## Slack Context

Gorkie does not yet preload bounded Slack context like Claude.

Target behavior:

- thread mentions preload the latest 50 thread messages;
- channel mentions preload the latest 20 channel messages;
- forwarded thread behavior can have a larger cap later;
- the prompt must say exactly what was loaded and what was not loaded;
- reader tools stay available for older or broader context.

Privacy work must happen first for DM/private-channel reads. The bot can read DMs it has token access to, which is both useful and dangerous.
